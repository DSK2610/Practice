Forms

The < form > tag is used to create an HTML form for user input.
The < form > element can contain one or more of the following form elements:

    < input text >
    < input radio for selecting one of many choices> Black White
    < input checkbox >
    < button >
    < select >
    < submit >
    < option >
    < optgroup >
    < fieldset >
    < label >
